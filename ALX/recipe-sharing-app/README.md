# Recipe Sharing Application

This is a simple Recipe Sharing Application built with React and Zustand for state management. Users can view a list of recipes and add new recipes to the collection.

## Features

- Display a list of recipes with titles and descriptions.
- Add new recipes through a form.

## Technologies Used

- React
- Zustand
- TypeScript

## Getting Started

To get a local copy up and running, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/recipe-sharing-app.git
   ```

2. Navigate to the project directory:
   ```
   cd recipe-sharing-app
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and go to `http://localhost:3000` to see the application in action.

## Usage

- Use the form to add new recipes by entering a title and description.
- View the list of recipes displayed below the form.
