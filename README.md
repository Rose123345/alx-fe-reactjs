# alx-fe-reactjs

This repository contains the React project created for the ALX Frontend ReactJS task.

## Project
- Directory: `alx-react-app` (created with Vite + React)

## Quick start
```bash
cd alx-react-app
npm install
npm run dev
# open http://localhost:5173 in your browser
```

## Notes
- The app uses Vite and starts on port 5173 by default.
- See `alx-react-app/README.md` for template details.
